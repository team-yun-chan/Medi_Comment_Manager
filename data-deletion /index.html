데이터 삭제 정책 (Data Deletion Policy)

최종 업데이트: 2025년 1월 1일

1. 개요

Social Comment Manager는 사용자의 "잊혀질 권리"를 존중합니다. 본 문서는 사용자가 개인정보 및 계정 데이터를 삭제하는 방법을 설명합니다.

2. 삭제 가능한 데이터

  • 사용자 계정 정보 (이름, 이메일, 프로필)
  • Google (YouTube) 연동 정보 및 댓글 데이터
  • Meta (Instagram) 연동 정보 및 댓글 데이터
  • OAuth 액세스 토큰
  • 모더레이션 규칙 및 설정
  • 감사 로그 (90일 이내)

3. 데이터 삭제 방법

방법 1: 대시보드에서 직접 삭제 (권장)

  1. 로그인 후 대시보드 접속
  2. 설정 → 계정 관리
  3. "계정 삭제" 버튼 클릭
  4. 확인 후 "DELETE" 입력
  5. 최종 삭제 버튼 클릭

처리 기간: 즉시 비활성화, 7일 이내 완전 삭제

방법 2: 이메일 요청

이메일: support@yourdomain.com

제목: 계정 삭제 요청 (Data Deletion Request)

본문 포함 사항:

  • 이름
  • 등록된 이메일 주소
  • 요청 사유 (선택)

처리 기간: 7 영업일 이내

방법 3: Google/Meta에서 직접 삭제

Google:

  1. Google 계정 설정 접속
  2. "타사 앱 액세스 권한" → "Social Comment Manager" 찾기
  3. "액세스 권한 삭제" 클릭

Meta:

  1. Facebook 설정 접속
  2. "비즈니스 통합" → "Social Comment Manager" 찾기
  3. "삭제" 클릭

⚠️ 주의: 이 방법은 OAuth 권한만 철회합니다. 우리 서버의 데이터는 방법 1 또는 2로 별도 삭제 요청하세요.

4. 삭제 타임라인

시간 작업
즉시 (0일) OAuth 토큰 삭제, 로그인 세션 무효화, 계정 비활성화
7일 이내 사용자 정보, 연동 계정, 댓글 데이터 완전 삭제
30일 이내 백업 데이터, 로그 데이터 완전 삭제

5. 삭제되지 않는 데이터

  • YouTube/Instagram 원본 댓글 (원본 플랫폼에서 직접 삭제 필요)
  • 익명화된 통계 데이터 (개인 식별 불가)
  • 법적 의무로 보관해야 하는 데이터

6. 데이터 복구

  • 7일 이내: 복구 요청 가능 (support@yourdomain.com)
  • 7일 이후: 복구 불가능 (완전 삭제됨)

7. 연락처

데이터 삭제 관련 문의:

  • 이메일: support@yourdomain.com
  • 제목: 데이터 삭제 문의 (Data Deletion Inquiry)
  • 응답 시간: 영업일 기준 48시간 이내
